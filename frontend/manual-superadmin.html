Manual del Superadministrador

Guía completa para gestionar negocios, usuarios, contratos y configuración del sistema BUNNYDJPOS.

Versión 1.0 — Junio 2026
🔐

Acceso al sistema

Cómo ingresar al panel de superadministrador

Abrir el navegador

Ingresa a la URL del sistema: http://tu-dominio/superadmin

Ingresar credenciales

Escribe tu correo de superadmin y contraseña. Las credenciales iniciales se configuran en el servidor.

Ingresar

Haz clic en Ingresar. Si las credenciales son correctas, serás redirigido al Dashboard.

⚠️ El panel de superadmin es exclusivo para el administrador del sistema. No compartas estas credenciales con administradores de negocios.
📊

Negocios

Registro y administración de establecimientos

Desde aquí puedes crear, editar, activar o desactivar los negocios que usan el sistema.

Crear un nuevo negocio

Haz clic en "＋ Nuevo negocio"

Botón ubicado en la parte superior derecha del panel.

Completa los datos

Nombre del negocio, tipo (restaurante, peluquería, minimercado), dirección y teléfono.

Selecciona el tipo

El tipo determina qué módulos POS tendrá disponibles el negocio.

País (moneda y zona horaria)

Elige el país del negocio — fija automáticamente su moneda (ej. COP, CRC) y zona horaria para que caja y reportes calculen "hoy" correctamente.

Idioma mandarín (opcional)

El negocio siempre usa español. Activa la casilla "Habilitar 中文 (Mandarín)" solo si ese negocio necesita menú/interfaz también en mandarín.

Guardar

El negocio queda creado con estado pendiente hasta que se le asigne un contrato.

Acciones disponibles por negocio

BotónFunción
✏️ EditarModifica nombre, tipo o datos de contacto del negocio
🔲 QRGenera código QR con la URL del portal de reservas o empleados
✅ ActivarHabilita el acceso al sistema para ese negocio
🚫 DesactivarBloquea el acceso sin eliminar los datos
🗑 EliminarElimina el negocio permanentemente (no se puede deshacer)

Estados de un negocio

EstadoSignificado
activoEl negocio puede iniciar sesión y usar el sistema
pendienteRegistrado pero aún sin contrato o en proceso
inactivoAcceso bloqueado temporalmente
💡 El código QR es útil para pegar en el local del cliente — así sus clientes pueden escanear y hacer reservas o ver el menú directamente.

Usuarios

Gestión de cuentas de superadministradores

Este módulo gestiona los usuarios con acceso al panel de superadmin. No confundir con los usuarios de cada negocio (esos se gestionan desde el Admin de cada negocio).

➕

Crear usuario

Agrega un nuevo superadmin con nombre, correo y contraseña

✏️

Editar

Cambia nombre, correo o contraseña de un superadmin existente

🗑

Eliminar

Borra la cuenta del superadmin (no se puede deshacer)

⚠️ No elimines el único superadmin del sistema — perderías el acceso completo al panel.

Planes

Configuración de planes de suscripción

Define los planes que ofreces a tus clientes (negocios). Cada plan tiene un precio y características.

Crear plan

Define nombre, precio mensual, descripción y límites del plan.

Asignar a contrato

Al crear un contrato, selecciona el plan correspondiente.

Actualizar precio

Puedes editar el precio del plan — los contratos existentes no se modifican automáticamente.

Comisión de BunnyDJPOS por plan

Además del precio fijo del plan, el sistema calcula una comisión sobre las ventas del negocio, según su plan:

PlanComisión sobre ventas
Free (demo)0% — sin comisión, solo para pruebas
Basic9%
Plus7%
Premium5%
💡 Esta comisión es la que se muestra como Comisión BunnyDJ en el Dashboard y en Facturación — se calcula automáticamente, no requiere configuración adicional.
📄

Contratos

Gestión de acuerdos de servicio con los negocios

Un contrato vincula un negocio con un plan y un período de vigencia. El sistema activa automáticamente el negocio cuando llega la fecha de inicio.

Crear un contrato

Selecciona el negocio

El negocio al que vas a vincular el contrato.

Elige el plan

Selecciona el plan de suscripción contratado.

Define las fechas

Fecha de inicio y fecha de fin del contrato.

Valor y notas

Precio acordado y cualquier observación relevante.

Guardar

El número de contrato se genera automáticamente (ej: CONT-2026-001).

Estados de contratos

EstadoSignificado
pendienteContrato creado pero la fecha de inicio no ha llegado
activoContrato vigente, negocio con acceso completo
vencidoFecha de fin superada, el negocio puede quedar bloqueado
canceladoContrato anulado manualmente
💡 El sistema activa automáticamente el negocio cuando la fecha de inicio del contrato llega. No necesitas hacerlo manualmente.
⚠️ Cuando un contrato vence, el negocio NO pierde sus datos — solo se bloquea el acceso hasta que se renueve.
