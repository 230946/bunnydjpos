Portal de acceso

El Portal es la pantalla principal de cada negocio en DJPOS. Desde aquí todo el personal accede a las distintas aplicaciones del sistema.

¿Qué es el portal?

El Portal de acceso es la página de entrada de cada negocio. Al abrir la URL del negocio (con el parámetro ?n=NEGOCIO_ID), el sistema muestra las tarjetas de acceso disponibles según el tipo de negocio registrado.

No requiere inicio de sesión para verlo — cada tarjeta lleva a su propia pantalla de autenticación.

Cómo acceder

El administrador comparte la URL del portal con todo el personal. La URL tiene el formato:

http://IP-DEL-SERVIDOR/portal.html?n=ID_DEL_NEGOCIO

Al cargar la página, el sistema verifica automáticamente el negocio y muestra las tarjetas correspondientes.

Guarda la URL del portal como favorito en el navegador para acceder rápidamente.

Tarjetas disponibles

El portal muestra entre 2 y 4 tarjetas dependiendo del tipo de negocio:

🛡️ Administración
💻 POS
📅 Reservar cita
👤 Portal Empleado

Tarjeta Administración

Acceso al panel de administración del negocio. Requiere usuario y contraseña con rol de administrador o con los permisos correspondientes.

Dependiendo del tipo de negocio abre:

  • Restaurante / Bar → Admin Restaurante
  • Peluquería → Admin Peluquería
  • Minimercado → Admin Minimercado

Tarjeta POS

Acceso al punto de venta para cajeros y meseros. Requiere usuario y contraseña.

Dependiendo del tipo de negocio abre:

  • Restaurante / Bar → POS Restaurante
  • Peluquería → POS Peluquería
  • Minimercado → POS Minimercado

Tarjeta Reservar cita

Solo disponible para negocios tipo Peluquería. Permite que los clientes reserven citas desde su celular o computador sin necesidad de llamar.

No requiere usuario ni contraseña — cualquier persona con el enlace puede reservar.

Comparte el enlace del portal con los clientes para que reserven por su cuenta.

Tarjeta Portal Empleado

Solo disponible para negocios tipo Peluquería. Permite a cada empleado consultar su agenda del día y su horario semanal usando su número de cédula.

Alertas de contrato

El portal puede mostrar avisos relacionados con el estado del contrato del negocio:

  • ⚠ Por vencer — el contrato vence próximamente. El acceso sigue funcionando. El administrador debe renovar.
  • 🔒 Suspendido — el contrato está vencido o suspendido. Las tarjetas de acceso se ocultan y no se puede usar el sistema.
Si ves la pantalla de acceso suspendido, comunícate con el administrador del sistema para renovar el contrato.
